Programming Set
  1. WAp that converts object of celsius type to object of Farenheit type using the concept of Type Conversion
  2. WAP a class that represents the distance class and overload ++ and -- operator to increment and decrement distance.
  3. WAP to overload the relational operators to compare the length (in meters and centimeters) of two object
  4. WAP to create class named City having two member variables CityName (char[20]) and DistFromKtm(float). Add member functions to set and retrieve the CityName and DistFromKtm seperately. Add operator overloading to find the distance between the cities (just find the difference of DistFromKtm) and sum of distances of those cities from Kathmandu. In the main function, initialize three city objects. Set the first and second city to be Pokhara and Dhangadi. Display the sum of DistFromKtm of Pokhara and Dhangadi and Distance between Pokhara and Dhangadi
  5. Develop a program using a class with 3x3 matrix as data member. Overload the * Operator so as to multiply 3x3 matrices.
  6. Create a class having an array as mermber.Overload index operator([]) to input and display the elements in the array.
  7. WAP to add, subtract and multiply two matrices
  8. WAP to overload insertion (<<) and extraction (>>) operators.
